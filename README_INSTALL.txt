GEARBOX WORKSHOP - INSTALLATION INSTRUCTIONS
============================================

Your application has been packaged securely. 

OPTION 1: QUICK RUN (PORTABLE)
------------------------------
1. Unzip "Gearbox-Setup-v1.0.0.zip" to any folder (e.g. C:\Gearbox).
2. Open the folder "gearbox-fintech-rebuilt-win32-x64".
3. Double-click "gearbox-fintech-rebuilt.exe" to launch.

OPTION 2: "INSTALL" TO PROGRAM FILES
------------------------------------
1. Create a folder "C:\Program Files\Gearbox Workshop".
2. Copy the CONTENTS of the unzipped folder into this new folder.
3. Right-click "gearbox-fintech-rebuilt.exe" -> Send to -> Desktop (create shortcut).
4. You now have a Desktop shortcut!

TROUBLESHOOTING
---------------
- If Windows SmartScreen appears, click "More Info" -> "Run Anyway" (since this is a self-signed app).
- Ensure you have read/write access to the database file location if errors occur.
